# **CSI606-2025-02 - Turma 23 - Proposta de Trabalho Final**

## *Lucas Nunes Silveira*

### **Resumo**

O projeto consiste no desenvolvimento de uma aplicação web completa para planejamento e organização de viagens. A plataforma permite cadastrar uma viagem com informações essenciais como destino, datas e convidados, e oferece uma página principal onde o usuário pode gerenciar as atividades da viagem, links importantes e a lista de participantes. A aplicação será construída utilizando tecnologias modernas como React e Node.js, adotando arquitetura cliente-servidor e boas práticas de desenvolvimento web.

---

### **1. Tema**

O trabalho final tem como tema o desenvolvimento de um sistema web voltado para **organização de viagens**, permitindo que o usuário cadastre sua viagem e gerencie todas as informações relevantes de forma simples e intuitiva.

---

### **2. Escopo**

O projeto contempla as seguintes funcionalidades:

* **Cadastro de uma viagem**, contendo:

  * Destino (cidade)
  * Período da viagem
  * Lista de convidados
  * Botão de confirmação que direciona para a página principal da viagem

* **Página Home da viagem**, organizada em layout em grid com:

  * **Configurações da viagem** (local e data)
  * **Gerenciamento de Atividades**, divididas por dia:

    * Nome da atividade
    * Horário de início e término
    * Opção de marcar como concluída ou não concluída
    * Botão para cadastrar nova atividade
  * **Links Importantes**, onde o usuário pode adicionar e visualizar links relevantes (hotel, passagens, reservas, etc.)
  * **Sessão de Convidados**, exibindo a lista de pessoas adicionadas e permitindo incluir novos convidados

* **Tecnologias previstas**:

  * Front-end: React + Vite
  * Backend: Node.js + Express
  * Banco: PostgreSQL
  * Comunicação via API REST
  * Containerização com Docker

---

### **3. Restrições**

Não fazem parte do escopo:

* Autenticação ou sistema de login
* Suporte a múltiplos usuários independentes
* Upload de fotos ou arquivos
* Integrações com APIs externas
* Notificações, alertas ou lembretes automáticos
* Suporte a diversas viagens
